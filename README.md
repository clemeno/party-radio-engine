# party-radio-engine
Online Realtime Media Music Video Aggregator Radio Flux Manager

## Install

**External dependencies**
```
npm i
```

## Run

**prod pm2**
```
pm2 start pm2_host.config.js
```

**prod npm**
```
npm start
```

**test**
```
npm run test
```

**dev**
```
npm run dev
```
